Реализация нечеткого вывода для оценки шансов попадания в баскетбольную команду

15 Января, 2026

В рамках лабораторной работы была поставлена интересная задача — использовать библиотеку scikit-fuzzy на Python для моделирования реальной ситуации с помощью нечеткой логики.

Суть проекта

Модель предназначена для оценки шансов игрока попасть в сборную баскетбольную команду. В качестве входных параметров (антецедентов) мы берем такие метрики, как:

  • Рост кандидата
  • Выносливость
  • Процент успешных бросков

Реализация и код

Использование алгоритмов Мамдани и Сугено показало отличные результаты при настройке правил для нечеткого вывода. Ниже представлен базовый пример инициализации переменных:

import numpy as np
import skfuzzy as fuzz
from skfuzzy import control as ctrl

# Инициализация входных параметров
height = ctrl.Antecedent(np.arange(160, 220, 1), 'height')
stamina = ctrl.Antecedent(np.arange(0, 101, 1), 'stamina')

# Выходной параметр (шанс попасть в команду)
chance = ctrl.Consequent(np.arange(0, 101, 1), 'chance')

# Автоматическая генерация функций принадлежности
height.automf(3)
stamina.automf(3)

В итоге система выдает итоговый скоринг в процентах. Это отличный пример того, как математическое моделирование помогает тренеру или менеджеру принять объективное, взвешенное решение без жестких рамок классической бинарной логики.
